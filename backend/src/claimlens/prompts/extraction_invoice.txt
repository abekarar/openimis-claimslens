Extract the following fields from this hospital invoice/receipt:

Fields to extract:
- patient_name: Full name of the patient
- patient_id: Patient or insurance ID
- invoice_number: Invoice/receipt number
- invoice_date: Date of the invoice
- facility_name: Name of the healthcare facility
- line_items: List of services/items with description, quantity, unit_cost, total
- subtotal: Subtotal amount before tax
- tax_amount: Tax amount if applicable
- total_amount: Total invoice amount
- currency: Currency code
- payment_status: Paid, unpaid, or partial

Return a JSON object with key "fields" containing a list of objects:
{
  "fields": [
    {
      "field_name": "<name>",
      "value": "<extracted_value>",
      "confidence": <0.0-1.0>,
      "bounding_box": {"x": 0, "y": 0, "width": 0, "height": 0}
    }
  ]
}
